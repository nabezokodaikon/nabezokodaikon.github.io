Fast and smooth image viewer

PicSum is a Windows desktop image viewer that improves your browsing experience through GPU-accelerated rendering and heavily optimized file loading and startup performance.

Get it from Microsoft Store

Features

High-speed rendering with GPU

Smooth scrolling and viewing with minimized stutter.

Fast file loading and startup

Heavily tuned for quick startup and folder switching.

Spread (two-page) view

Display two pages/images side by side for comparison and layout checking.

Tab operations comparable to Chrome

Switch between multiple folders/views with tabs without interrupting your work.

Breadcrumb address bar

Navigate folder hierarchy quickly, similar to File Explorer.

Easy access to folders

Open frequently used folders and folder history with a single click.

Organize files with tags

Classify files with tags to help with organization and later search.

Supported formats

PicSum supports the following image formats:

  • Avif
  • Bmp
  • Gif
  • Heic
  • Heif
  • Ico
  • Jfif
  • Jpeg
  • Jpg
  • Png
  • Svg
  • Webp

To display AVIF files, the following extension is required.
AV1 Video Extension

To display HEIC and HEIF files, the following extension is required.
HEIF Image Extensions

Mouse actions

  • Close tab.Middle-click on tab
  • Switch tabs.Scroll wheel on tab
  • Open item in a new tab.Middle-click on item in file list
  • Open folder in a new tab.Middle-click on address bar item
  • Open home page in a new tab.Middle-click on Home button
  • Open starred page in a new tab.Middle-click on Star button
  • Open bookmarks page in a new tab.Middle-click on Bookmarks button
  • Open history page in a new tab.Middle-click on History button
  • Open page for the selected tag in a new tab.Middle-click on tag in tag list
  • Go backward/forward in page history.Side buttons

Keyboard shortcuts

  • Close tab.Control + W
  • Open new tab.Control + T
  • Open new window.Control + N
  • Scroll to bottom of file list.Shift + G
  • Scroll to top of file list.GG
  • Switch to next tab to the right.Control + Tab
  • Switch to previous tab to the left.Control + Shift + Tab
  • Go back in page history.Alt + Left Arrow
  • Go forward in page history.Alt + Right Arrow
