PicSum is an image viewer that combines ultra-fast GPU-accelerated rendering, a meticulously tuned startup time of under 0.2 seconds, and intuitive tab operations comparable to Google Chrome. Eliminate the stress of image browsing with overwhelming speed and familiar operations.

Features

Fast GPU-accelerated Rendering

Reduces display lag for a comfortable browsing experience.

Fast File Loading / Instant Startup

Meticulously tuned for smooth transitions from startup to switching images.

Support for Spread View

Display two pages or images side-by-side, convenient for comparison or layout checks.

Chrome-like Tab Operations

Switch between multiple folders or views with tabs without interrupting your workflow.

Breadcrumb Address Bar

Navigate through hierarchies like File Explorer to quickly reach your destination.

Easy Folder Access

Open folders quickly from favorite lists, browsing history, and more.

Organization with Tagging

Classify files with tags to help with organization and future searches.

Supported Formats

Supports the following image formats:

  • Avif AV1 Video Extension
  • Bmp
  • Gif
  • Heic / Heif HEIF Image Extensions
  • Ico
  • Jfif
  • Jpeg
  • Jpg
  • Png
  • Svg
  • Webp WebP Image Extensions

Mouse Operations

  • Close tab: Middle-click on tab
  • Switch tabs: Mouse wheel on tab
  • Open item in new tab: Middle-click on file list item
  • Open folder in new tab: Middle-click on address bar item
  • Open homepage in new tab: Middle-click on home button
  • Open star page in new tab: Middle-click on star button
  • Open bookmarks in new tab: Middle-click on bookmark button
  • Open history in new tab: Middle-click on history button
  • Open tag page in new tab: Middle-click on tag in tag list
  • Back / Forward in history: Side buttons

Keyboard Shortcuts

  • Close tab: Control + W
  • Open new tab: Control + T
  • Open new window: Control + N
  • Scroll to bottom of file list: Shift + G
  • Scroll to top of file list: GG
  • Switch to right tab: Control + Tab
  • Switch to left tab: Control + Shift + Tab
  • Go back in history: Alt + Left Arrow
  • Go forward in history: Alt + Right Arrow
  • Reload page: Control + R / F5
