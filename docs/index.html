特徴

フォルダ閲覧が速い

フォルダを開くとサムネイル一覧が表示され、矢印キーやクリックで素早く切り替えできます。

情報パネル

ファイルサイズ、撮影日時、解像度などを確認しながら選別できます。

お気に入り/選別

気に入った写真をマークして、後からまとめて見返せます。

扱いやすい操作

ズーム・回転・フルスクリーンなど、画像ビューアに欲しい基本操作を揃えています。

Windows向け

Windows 10/11 のデスクトップで動作するアプリとして、マウス/キーボード操作に最適化します。

軽量

起動・切り替え・拡大縮小が引っかからないことを重視した設計を目指します。

FAQ

HEICが開けないことがあります

Windows側の拡張（HEIF/HEVC コーデック）の有無によっては、HEICの表示ができない場合があります。 その場合は Microsoft Store の拡張導入、またはJPEGへの変換をご検討ください。

オフラインで使えますか？

デスクトップアプリとしてローカル画像を扱う想定のため、基本的にオフラインで利用できます。

このページはどこを編集すれば良いですか？

docs/index.html の「ダウンロード」セクションに配布URLを入れ、 スクリーンショットを docs/assets/screenshot.png に置いてください。
