画像ビューアに、
Chromeの操作感を。

PicSum は、GPUによる超高速描画と徹底的にチューニングされた0.2秒を切る高速起動、そして Google Chrome と遜色ない直感的なタブ操作を兼ね備えた 画像ビューアです。 圧倒的なスピードと馴染みのある操作性で、画像閲覧のストレスをゼロにします。

Microsoft Storeで入手

特徴

GPUを使用した高速な描画

表示の引っかかりを抑え、快適な閲覧ができます。

高速なファイル読み込み / 高速起動

徹底的にチューニングし、起動から切り替えまでをスムーズにします。

見開き表示に対応

2ページ/2枚を並べて表示でき、比較やレイアウト確認にも便利です。

GoogleChromeと遜色ないタブ操作

複数フォルダ/ビューをタブで切り替え、作業を中断しません。

パンくずリストのアドレスバー

エクスプローラーのように階層を辿れて、目的の場所へ素早く移動できます。

フォルダへのアクセスが簡単

よく開くフォルダの一覧、フォルダの閲覧履歴などからすぐに開けます。

タグ付けによるファイルの整理

タグによる分類で、ファイル整理や後からの検索に役立ちます。

画像形式の変換

画像ファイルを 'Avif', 'Gif', 'Jpg', 'Png', 'Webp' に変換することができます。

対応画像形式

次の形式に対応しています。

  • Avif AV1 ビデオ拡張機能
  • Gif
  • Heic / Heif HEIF 画像表示オプション
  • Ico
  • Jfif
  • Jpeg
  • Jpg
  • Png
  • Svg
  • Webp WebP 画像表示オプション

マウス操作

  • タブを閉じる。タブ上でミドルクリック
  • タブを切り替える。タブ上でマウスホイール
  • 新しいタブで項目を開く。ファイルリストの項目をミドルクリック
  • 新しいタブでフォルダを開く。アドレスバーの項目をミドルクリック
  • 新しいタブでホームページを開く。ホームボタンをミドルクリック
  • 新しいタブでスターページを開く。スターボタンをミドルクリック
  • 新しいタブでブックマークページを開く。ブックマークボタンをミドルクリック
  • 新しいタブで履歴ページを開く。履歴ボタンをミドルクリック
  • 新しいタブで選択したタグのページを開く。タグ一覧でタグをミドルクリック
  • ページの履歴を戻る / 進む。サイドボタン

キーボードショートカット

  • タブを閉じる。Control + W
  • 新しいタブを開く。Control + T
  • 新しいウィンドウを開く。Control + N
  • ファイルリストで最下段までスクロールする。Shift + G
  • ファイルリストで最上段までスクロールする。GG
  • 右のタブに切り替える。Control + Tab
  • 左のタブに切り替える。Control + Shift + Tab
  • ページの履歴を戻る。Alt + 左矢印
  • ページの履歴を進む。Alt + 右矢印
  • ページを再読み込み。Control + R / F5

ダウンロード

PicSum は Microsoft Store で配布しています。

Windows

ストアからインストールできます。

Microsoft Storeで開く
  • 対応OS: Windows 10/11
  • 配布: Microsoft Store

インストール手順

  1. Microsoft Store を開く
  2. 「入手」からインストール
  3. 起動してフォルダを開く

ストアの表示は環境により言語やボタン表示が異なる場合があります。
